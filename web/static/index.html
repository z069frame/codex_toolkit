Generate Payment Link

Generate ChatGPT checkout link. Leave email blank to auto-pick an unsubscribed account with valid AT. Use Redirect (old) for standalone Stripe URLs (no login needed).

💡 PayPal submit requires Country = DE (Stripe only offers PayPal to DE sessions — other countries get card-binding only). The PayPal billing locale is independent — best to match the server's outbound IP (US for Railway). Mismatched billing often triggers Stripe Radar. Default: generate pay-link with Country=DE, then PayPal submit with billing=US. PayPal HTTP calls go DIRECT (no proxy) — ProxySeller is blocked by Stripe. The PayPal auth URL is opened in your own browser manually (no cookies/automation).
